# Transport Route Planner

## Overview
Transport Route Planner is a Java-based project that calculates optimized urban transportation routes using structured JSON data.

## Features
- Route calculation
- Multi-transport support (bus, tram, taxi, walking)
- JSON-based data modeling
- Optimized path selection

## Technologies Used
- Java
- Maven
- JSON

## Purpose
Developed as an academic project to explore data modeling and route optimization techniques.
